# cm1utils

Python tools and workflows for analyzing output from the **Cloud Model 1 (CM1)**.

This package contains reusable utilities developed during my PhD for working with CM1 simulations, including routines for data processing, diagnostics, visualization, and workflow automation.

---

## Structure

cm1utils/
├── tools/        # Core utilities (math, plotting, data handling)  
└── workflows/    # Higher-level workflows for CM1 analysis and preprocessing  

The `tools` module contains general-purpose helper functions, while `workflows` provides higher-level routines tailored to common CM1 analysis tasks.

---

## Notes
This repository contains __code only__.
Data, figures, and exploratory notebooks are kept in a separate project directory.
